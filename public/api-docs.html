Developer API Documentation

Integrate ebotconnect directly into your software. You can programmatically check your bot's connection status and dispatch messages to your customers. Note: API Access requires the Premium subscription plan.

Authentication

All API endpoints are authenticated using an API key. You can generate your API key from the Developer API section in your dashboard.

Include your key in the header of every request:

x-api-key: ebot_your_api_key_here
warning

Keep your API key secure. Never share it publicly or expose it in client-side code.

GET

/api/v1/status

Check the current connection status of your WhatsApp bot session.

Example Request (cURL)

curl -X GET https://your-server.onrender.com/api/v1/status \
  -H "x-api-key: your_api_key_here"

Response

{
  "status": "connected",
  "connected_at": "2023-10-25T14:30:00.000Z"
}
POST

/api/v1/messages/send

Send a text message to a specific WhatsApp number. Ensure your bot is active and connected before sending.

JSON Body Parameters

Parameter Type Description
to String The recipient's phone number including country code (e.g., 2348000000000).
message String The text content of the message you want to send.

Example Request (cURL)

curl -X POST https://your-server.onrender.com/api/v1/messages/send \
  -H "Content-Type: application/json" \
  -H "x-api-key: your_api_key_here" \
  -d '{
    "to": "2348123456789",
    "message": "Hello from the ebotconnect API!"
  }'

Response

{
  "success": true,
  "message": "Message sent successfully"
}
